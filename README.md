# 1D-simple-ewallet-project
This is a simple e-wallet system designed to track account balances, complete with a login and register system, as well as simple balance deposit and withdrawing system within it.

To use it, simply run the code, which will create a .json file that contains data. This data persists across different runs.
Afterwards, it will show you numbers to choose from correlating to different functions. Register an account, and you will have 0 in your balance for now.
Then, log in. It will show you the main menu and your current balance.
Simply choose one of the different functions according to your purpose and your account balance will update accordingly.
There is also a logout choice should you want to login to another account or register another account. There is also an exit to completely exit the program.
